# OGP画像のダウンロード手順

## 推奨画像

以下のPexels画像をOGP画像として使用することをお勧めします：

### オプション1: 温かい雰囲気の人物画像
- URL: https://www.pexels.com/photo/three-women-standing-beside-wooden-table-3184291/
- 直接ダウンロード: https://images.pexels.com/photos/3184291/pexels-photo-3184291.jpeg?auto=compress&cs=tinysrgb&w=1200&h=630

### オプション2: 優しい笑顔の画像
- URL: https://www.pexels.com/photo/woman-in-white-crew-neck-t-shirt-smiling-3184418/
- 直接ダウンロード: https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&w=1200&h=630

## ダウンロード手順

1. 上記のURLのいずれかにアクセス
2. 画像をダウンロード（推奨サイズ: 1200x630px）
3. ファイル名を `ogp.png` または `ogp.jpg` に変更
4. このファイルを `public/` ディレクトリに配置
5. 既存の `ogp.png` ファイルを置き換える

## 画像編集（オプション）

OGP画像にテキストを追加する場合：
- サイト名: 優しさMAX
- キャッチコピー: 心が解ける優しい世界へ
- ロゴやブランドカラー（アイボリー、パステルピンク）を追加

## OGP画像の要件

- **推奨サイズ**: 1200x630px
- **最小サイズ**: 600x315px
- **アスペクト比**: 1.91:1
- **ファイル形式**: JPG, PNG, WebP
- **ファイルサイズ**: 8MB以下

## 確認方法

画像を配置後、以下のツールで確認：
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)

## 注意事項

- Pexelsの画像は商用利用可能で、クレジット表記は不要です
- ただし、写真家への感謝の気持ちとして、クレジット表記を追加することは推奨されます
- 画像を編集して使用することも可能です
